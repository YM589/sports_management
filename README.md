# 项目概述

本系统基于Django框架与MySQL数据库开发，面向田径运动会打造数字化管理平台，划分**管理员、裁判、运动员**三类角色，实现赛事管理、报名、成绩录入、排名核算、团队积分、公告发布等全流程线上化，简化赛事组织流程，提升数据管理效率与准确性。 



# 功能介绍 

1. **管理员**：管理赛事信息、用户账号与权限，导出数据报表，发布赛事公告。
1. **裁判**：查看负责项目信息，录入并校验运动员成绩，自动更新排名与团队积分。 
1. **运动员**：自主报名赛事，查看个人成绩与团队排名，可申请合理弃权。
1. **系统通用**：自动校验报名与成绩数据，实时计算积分排名，权限隔离保障安全。



# 技术栈

- **开发语言**：Python 3.10
- **后端框架**：Django 5.1.3
- **数据库**：MySQL 8.0
- **工具库**：openpyxl、pandas、matplotlib
